# GovHack 2025 競賽資訊整理

## 🎯 **競賽概述**
GovHack 是澳洲最大的政府開放數據創新競賽，參賽者使用政府開放數據來解決社會問題並創造創新解決方案。

## 🏆 **2025 年三大核心主題**

### 1. **Community Empowerment and Connection (社區賦權與連結)**
- **目標**: 建立解決方案和工具，以加強社會凝聚力
- **重點**: 賦予社區識別當地需求的能力
- **方法**: 透過數位技術將居民與重要服務連結起來

### 2. **Productivity and Innovation (生產力與創新)**
- **目標**: 構思並產生解決方案和策略
- **重點**: 加速數據和人工智慧的使用
- **方法**: 改進數據和 AI 的使用方式，提高生產力和經濟增長

### 3. **Digital Safety and Trust (數位安全與信任)**
- **目標**: 制定策略以增強數位安全和信任
- **重點**: 賦予社區和組織能力，安全地導航線上空間和數據
- **保護**: 保護弱勢群體免受網路威脅、錯誤資訊和 AI 造成的危害

## 📋 **參賽提交要求**

### **必須提交的內容**
1. **專案展示影片 (3分鐘)**
   - 格式：MP4 或 YouTube 連結
   - 內容：專案介紹、問題說明、解決方案、數據使用

2. **專案文檔**
   - `README.md`：專案概述、安裝說明、使用方法
   - 專案描述：詳細說明問題、解決方案、技術架構
   - 數據使用說明：列出所有使用的政府開放數據集

3. **可運行的專案**
   - Web 應用：網站、API、儀表板
   - 移動應用：iOS/Android 應用
   - 數據可視化：互動式圖表、地圖、分析工具
   - API 服務：數據處理或分析服務

### **不能提交的內容**
- 僅有概念或設計稿（必須可運行）
- 未使用政府開放數據的專案
- 違反競賽規則的內容
- 抄襲或侵權的專案

## 🔑 **數據使用要求**

### **強制要求**
- **必須使用至少一個官方數據集**才能獲得獎項資格
- 官方數據集列表可在 Hackerspace 上找到

### **允許使用的數據**
- GovHack 官方數據集
- 其他符合許可條款的開放數據集
- 公開可訪問的網路服務（免費）

### **禁止使用的數據**
- 私人和訂閱型 API
- 需要付費的數據服務

## 🎖️ **徽章系統**

### **可獲得的徽章**
- **數據使用徽章**：最大化政府數據利用
- **創新徽章**：獨特的解決方案
- **技術徽章**：技術實現質量
- **社會影響徽章**：實際問題解決

## 🚀 **NSW 專案建議**

### **推薦的系統類型**
1. **NSW 社區服務整合平台**
   - 主題覆蓋：社區賦權 + 數位安全
   - 功能：社區需求識別、服務匹配、安全驗證

2. **NSW 數據驅動決策平台**
   - 主題覆蓋：生產力創新 + 社區賦權
   - 功能：數據可視化、AI 分析、社區洞察

3. **NSW 數位安全社區網絡**
   - 主題覆蓋：數位安全 + 社區連結
   - 功能：安全資訊分享、威脅檢測、社區互助

## 💡 **獲勝策略關鍵點**

- **數據整合**：多源數據的創新組合
- **實用性**：解決真實的政府/社會需求
- **技術創新**：展示技術能力
- **影響力**：可量化的社會效益
- **創新性**：獨特的解決方案角度
